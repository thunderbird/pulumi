tb_pulumi Documentation
=======================

tb_pulumi is a library used to extend the features of the `Pulumi <https://www.pulumi.com/>`_ infrastructure as code
tool, and to define infrastructure patterns common to Thunderbird Pro Services.

.. seealso::

   The `live version of this page <https://thunderbird.github.io/pulumi/index.html>`_ is syncronized with the project's
   ``main`` branch. If you require documentation for a specific version of this library, instructions for generating it
   can be `found in the readme <https://github.com/thunderbird/pulumi/?tab=readme-ov-file#building-documentation>`_.
