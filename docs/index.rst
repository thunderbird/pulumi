tb_pulumi Documentation
=======================

tb_pulumi is a library used at Thunderbird to help developers bring cloud-based infrastructure
online to host the services they're developing using the `Pulumi <https://www.pulumi.com/>`_
infrastructure as code tool.

.. seealso::

   The `live version of this page <https://thunderbird.github.io/pulumi/index.html>`_ is syncronized with the project's
   ``main`` branch. If you require documentation for a specific version of this library, instructions for generating it
   can be `found in the readme <https://github.com/thunderbird/pulumi/?tab=readme-ov-file#usage>`_.
